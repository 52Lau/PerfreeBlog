Perfree

安装说明

欢迎使用Perfree,在开始前, Perfree 需要您提前安装好Mysql数据库,并提供以下信息:
1. 数据库用户名
2. 数据库密码
3. 数据库主机地址
4. 数据库端口号
如您已准备好此工作,点击下方"下一步"按钮即可开始安装

许可协议

Perfree基于GPL协议发布,我们允许用户在GPL协议许可的范围内使用,拷贝,修改和发布此程序,您可以自由的将其用于商业以及非商业用途中
