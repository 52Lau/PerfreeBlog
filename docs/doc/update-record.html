# 更新日志

  • 2021年11月2日 V1.2.6正式版发布 正式版

    新增 系统在线更新,无需操作服务器,在网站后台即可实现更新(如使用Nginx代理,则需要开启Nginx的websocket支持,参考文档->常见问题6)
    优化 安装时检测数据库是否存在,可选择重新安装和跳过数据库安装

  • 2021年10月25日 V1.2.5正式版发布 正式版

    新增 文章编辑器新增附件选择
    优化 后台样式优化
    优化 后台响应式处理

  • 2021年9月15日 V1.2.4正式版发布 正式版

    新增 更细化的权限控制->管理员, 文章编辑,文章贡献, 普通用户,访问用户,具体可参考文档->权限控制
    优化 优化更新版本时,更新sql的执行逻辑

  • 2021年9月14日 V1.2.3正式版发布 正式版

    修复 swagger接口文档number exception问题
