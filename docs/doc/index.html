# 简介

PerfreeBlog是一款基于java开发的博客建站平台,后端采用SpringBoot (opens new window)、Shiro (opens new window)、Mybatis (opens new window)、Enjoy (opens new window)模板引擎等技术进行开发,给您带来全新的创作体验

目前PerfreeBlog的所有功能均为个人开发,因能力有限,其中很多技术都是边学习边使用的,所以有些地方可能有不足之处,欢迎各位共同交流,同时如若本项目对您有所帮助,请为它点赞 (opens new window)支持

# 特性

  • 设计简洁，界面美观
  • 采用Markdown (opens new window)编辑器,支持一键插入视频、图片
  • 支持多主题自由切换
  • 主题在线编辑
  • 支持扩展插件
  • 友情链接管理
  • 支持附件管理
  • 主题开发简单快速
  • 安装部署简单
  • 支持在线更新
  • 支持mysql (opens new window)/sqlite (opens new window)数据库
