    # 常见问题汇总

    # 1. 访问的用户如何修改头像

    项目默认采用https://en.gravatar.com进行头像处理,访问的用户可去https://en.gravatar.com进行修改头像

    # 2. 如何自定义页面

    如主题中附带page.html,可在后台管理->页面管理->新增页面,之后去菜单管理->新增菜单->绑定页面即可

    # 3. 如何配置nginx

     server {
        listen       80;
        server_name  www.yinpengfei.com;
        gzip  on;
        gzip_disable "MSIE [1-6].";
        client_max_body_size    1000m;
        location / {
                proxy_redirect off;
                proxy_set_header Host $host;
                proxy_set_header X-Forwarded-Host $host;
                proxy_set_header X-Forwarded-Server $host;
                proxy_set_header X-Real-IP $remote_addr;
                proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
                proxy_buffering on;
                    proxy_pass http://127.0.0.1:8080;
        }
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }

    }

    # 4.如何安装及更新主题

    安装主题可在后台管理->主题管理->所有主题中安装,更新主题可先切换至其他主题,然后卸载要更新的主题,安装即可
