# 下载

本教程以 CentOS 7.x 为例，其他 Linux 发行版基本一致。

首先将PerfreeBlog下载至本地备用,以便后续步骤,Linux推荐下载tar.gz安装包,Windows推荐下载zip安装包

  • 下载地址一: github https://github.com/perfree/PerfreeBlog/releases (opens new window)
  • 下载地址二: gitee https://gitee.com/perfree/PerfreeBlog/releases (opens new window)

# 环境配置

请确保服务器的软件包为最新

sudo yum update -y
1

安装java运行环境(已安装的请忽略)

# 安装 OpenJRE
sudo yum install java-1.8.0-openjdk -y

# 检测是否安装成功
java -version

PerfreeBlog支持mysql和sqlite数据库,推荐数据库为mysql数据库,可参考MySQL 安装 (opens new window),当然也可以选择使用sqlite数据库,无需安装

# 安装PerfreeBlog

将下载的Perfree安装包上传至服务器,解压压缩包

 tar zxvf perfree-web-xxx.tar.gz
1

进入解压后的目录

cd perfree-web
ls
1
2

将会看到目录内有如下文件

# 启动

./start.sh start
1

如遇到-bash: ./start.sh: /bin/bash^M: 坏的解释器: 没有那个文件或目录错误,则执行以下命令

sed -i 's/\r//' ./start.sh
./start.sh start
1
2

执行完毕会在终端显示如下信息

常用命令

# 运行
./start.sh start

# 查看运行状态
./start.sh status

# 重新启动
./start.sh restart

# 停止
./start.sh stop

# 访问

默认端口8080,如访问不到请查看服务器防火墙是否开启8080端口的访问权限,如需更改默认端口请查看更换端口

启动完成后,访问http://服务器ip:8080进入安装页,按照提示进行配置即可
