# 介绍

全局变量,指在任意页面都可使用,如#(SEO_TITLE??)

名称 类型 描述
SEO_TITLE 字符串 网站SEO标题
SEO_KEYWORD 字符串 网站SEO关键字
SEO_DESC 字符串 网站SEO描述
WEB_TITLE 字符串 网站标题
WEB_META_KEYWORD 字符串 网站关键字
WEB_META_DESC 字符串 网站描述
loginUser User对象 当前登录用户

# SEO_TITLE

用于输出网站SEO标题

#(SEO_TITLE??)
1

# SEO_KEYWORD

用于输出网站SEO关键字

#(SEO_KEYWORD??)
1

# SEO_DESC

用于输出网站SEO描述

#(SEO_DESC??)
1

# WEB_TITLE

用于输出网站标题

#(WEB_TITLE??)
1

# WEB_META_KEYWORD

用于输出网站关键字

#(WEB_META_KEYWORD??)
1

# WEB_META_DESC

用于输出网站描述

#(WEB_META_DESC??)
1

# loginUser

当前登录的用户,如果未登录则为空

#if(loginUser)
    #(loginUser.userName)已登录
#else
   未登录
#end

常用属性描述:

名称 类型 描述
loginUser.userName 字符串 用户名
loginUser.id 字符串 用户id
loginUser.account 字符串 账户
loginUser.status 数字 状态0正常,1禁用
loginUser.avatar 字符串 头像地址
loginUser.website 字符串 网站地址
loginUser.email 字符串 邮箱地址
loginUser.hasAdmin() 字符串 是否为管理员
